تعلم HTML و CSS في أسبوع: الدليل العملي للمبتدئين

HTML و CSS هما أساس كل موقع على الإنترنت. تستطيع تعلمهما في أسبوع واحد إذا اتبعت الخطة الصحيحة.

اليوم الأول والثاني: HTML

HTML هي لغة هيكل الصفحة. تخيلها كهيكل عظمي للجسم. أهم الوسوم: h1-h6، p، a، img، div، span.

اليوم الثالث والرابع: CSS الأساسي

CSS هي التصميم والألوان. ابدأ بتعلم: color، background، font-size، margin، padding، border.

اليوم الخامس: CSS Flexbox

Flexbox يحل مشكلة توزيع العناصر. بسطرين من الكود تستطيع توسيط أي شيء بشكل مثالي.

اليوم السادس والسابع: مشروع حقيقي

ابنِ صفحة شخصية تحتوي على اسمك وصورتك ومهاراتك. هذا أفضل طريقة لترسيخ ما تعلمته.
