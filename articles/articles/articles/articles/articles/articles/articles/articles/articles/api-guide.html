ما هو الـ API وكيف يعمل؟ شرح بسيط بأمثلة حقيقية

API كلمة تسمعها كثيراً في عالم التقنية لكن قليلون يفهمون ما تعنيه فعلاً. في هذا المقال سنشرحها بطريقة لن تنساها.

تشبيه المطعم

تخيل مطعماً: أنت الزبون، المطبخ هو الخادم، والنادل هو الـ API. أنت لا تدخل المطبخ مباشرة — النادل يأخذ طلبك ويوصله للمطبخ ثم يعود بالطعام. الـ API يفعل نفس الشيء بين التطبيقات.

أمثلة حقيقية على API

  • تسجيل الدخول بـ Google في تطبيقات أخرى
  • خرائط Google مدمجة في تطبيق Uber
  • دفع PayPal في متاجر إلكترونية
  • أحوال الطقس في تطبيق الجوال

لماذا يهم المطور؟

بدلاً من بناء كل شيء من الصفر، تستخدم APIs جاهزة. تريد إضافة الدفع لموقعك؟ استخدم Stripe API. تريد إرسال إيميلات؟ استخدم SendGrid API. توفر وقتاً وجهداً هائلاً.
